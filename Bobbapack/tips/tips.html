Tips & Tricks

This page gives Habbos some useful tips & tricks that can be used in the hotel.

Alt Codes

To use one of these pictures in your dialogue hold down alt (or shift for the heart) and press the corresponding number on the number pad at the right of the keyboard. If you install the Habbo font you will be able to see all of these alt codes on the character map. Some codes may use a different number combination but still get the same result.

Chooser Commands

These commands work only for Habbos who are in Habbo Club.


:chooser brings up the Chooser box. This gives the user information on who is in the room.


:furni brings up the Furniture Chooser box. This give the user information on what furni is in the room.
